第52章：测试数据生成

学习目标

  • 掌握AI生成测试数据的方法与类型
  • 理解边界值数据、异常数据的设计
  • 学会生成Mock数据与测试数据集
  • 了解数据隐私保护的脱敏处理

测试数据类型

数据类型生成要点AI优势
正向数据符合规格的有效数据快速生成大量样本
边界数据最大/最小/临界值自动识别边界条件
异常数据非法格式、空值、溢出覆盖多种异常场景
Mock数据模拟API响应生成真实感数据

AI生成测试数据示例

Python
# AI生成的用户注册测试数据
test_users = [
    # 正向数据
    {"username": "user001", "email": "user001@test.com", "age": 25},
    {"username": "user002", "email": "user002@test.com", "age": 30},
    
    # 边界数据
    {"username": "a", "email": "a@b.com", "age": 0},       # 最小长度
    {"username": "maxlenuser1234567890", "email": "max@test.com", "age": 150},  # 最大
    
    # 异常数据
    {"username": "", "email": "", "age": -1},            # 空值
    {"username": "user!", "email": "invalid", "age": 999},  # 格式错误
]

本章小结

  • 测试数据类型：正向、边界、异常、Mock四类
  • AI优势：快速批量生成、自动识别边界、覆盖异常场景
  • 脱敏处理：生成测试数据时保护隐私信息
