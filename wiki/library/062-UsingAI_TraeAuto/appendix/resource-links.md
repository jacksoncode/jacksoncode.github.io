## 数据集资源

### 通用数据集平台

1. **Kaggle Datasets**
   - [官方网站](https://www.kaggle.com/datasets) - Kaggle竞赛平台上的数据集集合
   - [特点]：涵盖各种领域，包括计算机视觉、自然语言处理、结构化数据等

2. **Hugging Face Datasets**
   - [官方网站](https://huggingface.co/datasets) - Hugging Face提供的NLP和多模态数据集
   - [特点]：与Transformers库无缝集成，支持流式加载

3. **Google Dataset Search**
   - [官方网站](https://datasetsearch.research.google.com/) - Google的数据集搜索引擎
   - [特点]：可以搜索来自全球各地的公开数据集

4. **UCI Machine Learning Repository**
   - [官方网站](https://archive.ics.uci.edu/ml/index.php) - UCI机器学习仓库
   - [特点]：包含大量经典机器学习数据集

### 计算机视觉数据集

1. **ImageNet**
   - [官方网站](http://www.image-net.org/) - 大规模图像识别数据集
   - [特点]：包含超过1400万张图像，分为2万多个类别

2. **COCO (Common Objects in Context)**
   - [官方网站](https://cocodataset.org/) - 目标检测、分割和字幕数据集
   - [特点]：包含33万张图像，标注了80个物体类别

3. **CIFAR-10/CIFAR-100**
   - [官方网站](https://www.cs.toronto.edu/~kriz/cifar.html) - 小型图像分类数据集
   - [特点]：CIFAR-10包含10个类别，CIFAR-100包含100个类别，每个类别有6000张图像

4. **MNIST**
   - [官方网站](http://yann.lecun.com/exdb/mnist/) - 手写数字识别数据集
   - [特点]：包含7万张28×28像素的手写数字图像

5. **PASCAL VOC**
   - [官方网站](http://host.robots.ox.ac.uk/pascal/VOC/) - 视觉对象类别数据集
   - [特点]：包含20个对象类别，用于目标检测和分割任务

### 自然语言处理数据集

1. **GLUE (General Language Understanding Evaluation)**
   - [官方网站](https://gluebenchmark.com/) - 自然语言理解基准数据集
   - [特点]：包含9个不同的NLP任务，用于评估模型的通用语言理解能力

2. **SQuAD (Stanford Question Answering Dataset)**
   - [官方网站](https://rajpurkar.github.io/SQuAD-explorer/) - 斯坦福问答数据集
   - [特点]：包含10万个问答对，用于评估机器阅读理解能力

3. **IMDb (Internet Movie Database)**
   - [官方网站](https://www.imdb.com/) - 电影评论情感分析数据集
   - [特点]：包含5万条电影评论，每条评论带有正面或负面的情感标签

4. **WikiText**
   - [官方网站](https://blog.einstein.ai/the-wikitext-long-term-dependency-language-modeling-dataset/) - 基于维基百科的语言建模数据集
   - [特点]：包含长篇文本，适合训练处理长距离依赖的语言模型

5. **中文NLP数据集**
   - [CLUE (Chinese Language Understanding Evaluation)](https://www.cluebenchmarks.com/) - 中文语言理解测评基准
   - [THUCNews](http://thuctc.thunlp.org/) - 清华大学新闻分类数据集
   - [NLPCC (Natural Language Processing and Chinese Computing)](http://tcci.ccf.org.cn/conference/2021/dldoc/evadata.php) - 自然语言处理与中文计算会议数据集

### 音频与语音数据集

1. **LibriSpeech**
   - [官方网站](http://www.openslr.org/12/) - 有声读物语音识别数据集
   - [特点]：包含1000小时的英语朗读音频，用于训练语音识别系统

2. **Common Voice**
   - [官方网站](https://commonvoice.mozilla.org/) - Mozilla的多语言语音数据集
   - [特点]：包含多种语言的语音样本，由社区贡献

3. **MUSDB18**
   - [官方网站](https://sigsep.github.io/datasets/musdb.html) - 音乐源分离数据集
   - [特点]：包含150首专业制作的音乐曲目，每首曲目都有单独的音轨

4. **ESC-50**
   - [官方网站](https://github.com/karoldvl/ESC-50) - 环境声音分类数据集
   - [特点]：包含50个类别的环境声音，每个类别有40个样本

### 多模态数据集

1. **MSCOCO**
   - [官方网站](https://cocodataset.org/) - 同时包含图像和文本描述的数据集
   - [特点]：用于图像描述生成、图像-文本检索等多模态任务

2. **Flickr30k**
   - [官方网站](https://shannon.cs.illinois.edu/DenotationGraph/) - Flickr图像和文本描述数据集
   - [特点]：包含3万张图像，每张图像配有5个描述句子

3. **Conceptual Captions**
   - [官方网站](https://ai.google.com/research/ConceptualCaptions/) - Google的图像-文本对数据集
   - [特点]：包含330万张图像和对应的自然语言描述

4. **VGGFace2**
   - [官方网站](http://www.robots.ox.ac.uk/~vgg/data/vgg_face2/) - 大规模人脸识别数据集
   - [特点]：包含9131个身份的331万张图像
