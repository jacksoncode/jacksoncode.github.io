# Linux快捷键大全

## 1 Bash Shell快捷键

### 1.1 光标移动

| 快捷键 | 描述 | 说明 |
|--------|------|------|
| `Ctrl + A` | 移动到行首 | 快速跳到命令开头 |
| `Ctrl + E` | 移动到行尾 | 快速跳到命令结尾 |
| `Ctrl + B` | 向后移动一个字符 | 相当于左箭头 |
| `Ctrl + F` | 向前移动一个字符 | 相当于右箭头 |
| `Alt + B` | 向后移动一个单词 | 按单词移动 |
| `Alt + F` | 向前移动一个单词 | 按单词移动 |
| `Ctrl + XX` | 在行首和当前位置切换 | 快速切换 |

### 1.2 文本编辑

| 快捷键 | 描述 | 说明 |
|--------|------|------|
| `Ctrl + U` | 删除到行首 | 删除光标前的所有字符 |
| `Ctrl + K` | 删除到行尾 | 删除光标后的所有字符 |
| `Ctrl + W` | 删除前一个单词 | 删除光标前的一个单词 |
| `Alt + D` | 删除后一个单词 | 删除光标后的一个单词 |
| `Ctrl + D` | 删除当前字符 | 相当于Delete键 |
| `Backspace` | 删除前一个字符 | 标准退格键 |
| `Ctrl + H` | 删除前一个字符 | 相当于Backspace |
| `Ctrl + T` | 交换当前字符和前一个字符 | 交换两个字符 |
| `Alt + T` | 交换当前单词和前一个单词 | 交换两个单词 |

### 1.3 历史命令

| 快捷键 | 描述 | 说明 |
|--------|------|------|
| `Ctrl + R` | 反向搜索历史 | 搜索之前执行过的命令 |
| `Ctrl + P` | 上一个命令 | 相当于上箭头 |
| `Ctrl + N` | 下一个命令 | 相当于下箭头 |
| `Alt + .` | 插入上一个命令的最后一个参数 | 快速重用参数 |

## 9 自定义快捷键

### 9.1 查看当前快捷键

```bash
# 查看bash快捷键
bind -p

# 查看readline配置
cat ~/.inputrc

# 查看vim快捷键
:help index

# 查看screen快捷键
Ctrl + A + ?

# 查看tmux快捷键
Ctrl + B + ?
```

### 9.2 自定义Bash快捷键

```bash
# 编辑个人readline配置
cat >> ~/.inputrc << 'EOF'
# 自定义快捷键
"\C-x\C-l": "ls -la\n"          # Ctrl+X, Ctrl+L 执行ls -la
"\C-x\C-r": "ranger\n"         # Ctrl+X, Ctrl+R 启动ranger
"\C-x\C-g": "git status\n"     # Ctrl+X, Ctrl+G git状态
"\C-x\C-d": "df -h\n"          # Ctrl+X, Ctrl+D 显示磁盘空间
EOF

# 重新加载配置
bind -f ~/.inputrc
```

### 9.3 自定义Vim快捷键

```vim
" 添加到 ~/.vimrc
" 自定义快捷键
nnoremap <C-s> :w<CR>                    " Ctrl+S 保存
nnoremap <C-q> :q<CR>                    " Ctrl+Q 退出
nnoremap <C-z> u                         " Ctrl+Z 撤销
nnoremap <C-y> <C-r>                     " Ctrl+Y 重做
nnoremap <C-f> /                         " Ctrl+F 搜索
nnoremap <C-h> :%s///g<Left><Left><Left> " Ctrl+H 替换

" 快速保存和退出
nnoremap <leader>w :w<CR>
nnoremap <leader>q :q<CR>
nnoremap <leader>x :x<CR>

" 快速移动
nnoremap J 5j
nnoremap K 5k
nnoremap H 0
nnoremap L $
```

### 9.4 自定义Tmux快捷键

```bash
# 添加到 ~/.tmux.conf
# 修改前缀键
set-option -g prefix C-a
unbind-key C-b
bind-key C-a send-prefix

# 快速创建和切换窗口
bind-key c new-window -n default
bind-key C-c new-window -n "$(basename $(pwd))"
bind-key C-n next-window
bind-key C-p previous-window

# 快速分屏
bind-key | split-window -h -c "#{pane_current_path}"
bind-key - split-window -v -c "#{pane_current_path}"

# 快速调整面板大小
bind-key -r H resize-pane -L 5
bind-key -r L resize-pane -R 5
bind-key -r J resize-pane -D 5
bind-key -r K resize-pane -U 5

# 重新加载配置
```

**文件操作组合**：
- `Ctrl + A` → `Ctrl + K`：快速清空整行
- `Alt + B` → `Alt + D`：快速删除单词
- `Ctrl + R` → 输入关键词：快速搜索历史命令

- `Ctrl + U` → `Ctrl + Y`：剪切粘贴整行

- `dd` → `p`：剪切粘贴行
- `yy` → `P`：复制粘贴到上方
- `dw` → `i`：删除单词后插入
